CSS Design


1/4/2015

1. Margin is a clear area around the border, it impacts around the object. Border adds a line of however big of a px as indicated around the block. Padding is space within the block (so whereas margin is outside, padding is inside).

2. Classes can be used multiple times, but IDs can only be used once.

3. Fixed position is in relation to the browser. So if you fix a navigation bar to the top, it will stick to the top of the window when scrolling. Absolute will have a set location no matter what the other elements are. Overlapping can happen. It is almost like a trump. Relative is moving it from where it is originally placed.

4. Display inline will force everything to be in one line (it will widen if necessary), block allows the dimensions to be changed.
