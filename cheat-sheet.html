RUBY CHEAT SHEET
Hashes Stores information with a key and value, which would be called upon based on its key Hash_name = { key => value }

hash = { "name" => "John Doe",
"age" => "24",
"location" => "New York" }
Array Arrays are ordered, integer-indexed collections of any object. array_name = [array content]

array = [1,2,3,4]
array[0] = 1
String Holds and manipulates an arbitrary sequence of bytes, typically representing characters. tl;dr they are words/characters/numbers in quotation marks "this is a string"
Integers This class is the basis for the two concrete classes that hold whole numbers, Bignum and Fixnum. 23 and 4 are examples of Integers
